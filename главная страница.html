Как устроен мой сайт...



HTML CSS PHP JS
+ + - -
HTML - это структура сайта, его скелет. Без этого не может обойтись ни один сайт. CSS - это интерфейс сайта, графический дизайн, каскадные таблицы стилей. С помощью CSS мы можем менять цвет, шрифт, форму кнопки или таблицы, ориентировать текст в страничном пространстве. JavaScript - это язык программирования, применяющийся для добавления анимации, простейших скриптов и эффектов. PHP - это язык программирования, применяющийся для разработки web-сайтов. С помощью PHP мы регистрируемся, оставляем комментарии, загружаем и отправляем файлы.

Страницы сайтов — это набор текстовых файлов, размеченных на языке HTML. При загрузке этих файлов на компьютер пользователя, они обрабатываются и выводятся на экран. Язык HTML позволяет создавать и изменять различные элементы, добавлять новые функции, создавать гипертекстовые ссылки (гиперссылки) и вставлять в отображаемую страницу изображения, звукозаписи и другие мультимедийные элементы. Дизайн страницы можно изменить, добавив различные CSS стили или сценарии на языке JavaScript, с помощью которого можно сделать на станицу анимированой или добавив туда определенные действия. Страницы сайтов могут быть простым статичным набором файлов или создаваться специальной компьютерной программой на сервере. Они могут быть либо сделаны на заказ для отдельного сайта, либо быть готовым продуктом, рассчитанным на некоторый класс сайтов. Некоторые из них могут обеспечить владельцу сайта возможность гибкой настройки структурирования и вывода информации на веб-сайте. Такие управляющие программы называются системами управления содержимым (CMS). Каждый сайт рассчитан на разную аудиторию. Аудитории могут различаться по разным признакам: по языку, по возрасту, по виду используемого оборудования (некоторые сайты разделяются на мобильную и стандартную версию). Сайты могут иметь языковые версии (русскоязычная, англоязычная и другие).
